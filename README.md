# Crypto Dashboard

A professional-grade React + Tailwind CSS crypto price dashboard web application using CoinGeckoâ€™s public API for real-time crypto data.

## Features
- Live crypto price chart (react-chartjs-2)
- Top performing cryptos sidebar
- Portfolio section with mini charts
- Balance summary
- Search bar for coins
- Responsive dark UI with Tailwind CSS v3
- CoinGecko API integration

## Getting Started
1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```

## Tech Stack
- React
- Vite
- Tailwind CSS v3
- Chart.js
- CoinGecko API
- Heroicons
- ESLint & Prettier
